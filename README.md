## 🌟 Featured Projects

### Boston Housing Predictor
An end-to-end machine learning project that predicts housing prices using linear regression. The model is trained on two key features: number of rooms and LSTAT (percentage of lower-status population). Built with Python, scikit-learn, and deployed using Flask for real-time predictions.

**Technologies:** Python, scikit-learn, Flask, Linear Regression
